# Glossary

One line per term; the link points at the doc that defines it in depth. Where a term is a metaphor word, the parenthetical is its bound referent (the metaphor-referent binding rule, [concepts.md](concepts.md)).

- **AuditTrail (strand)** — provenance strand: first-seen timestamp plus a reference into the `ward.audit` event log. → [channels-and-strands.md](channels-and-strands.md#audittrail)
- **Channel** — (*axis of load a thread must hold under*): `Deliberate`, `Forced`, `Serialization`, or `Mutation`. → [concepts.md](concepts.md#channel--the-axis-of-load)
- **ContentHash (strand)** — hash commitment to a protected surface's content; structural floor on `Mutation` and `Forced`. → [channels-and-strands.md](channels-and-strands.md#contenthash)
- **Coven Cave** — the cockpit UI that will render weaves, threads, and strands, and host the proposal approval flow (Phase 4, not started). → [phases.md](phases.md#phase-4--coven-cave-ux-not-started)
- **`coven` daemon** — the shipped Rust trust boundary; every client is untrusted for enforcement purposes (`coven/docs/SAFETY-MODEL.md`). → [architecture.md](architecture.md#where-coven-threads-sits)
- **DegradeToProposal (verdict)** — the frayed-thread outcome: write staged to `~/.coven/pending/`, principal notified, protected surface untouched. → [authority-model.md](authority-model.md#degradetoproposal)
- **Deliberate (channel)** — familiar-initiated, principal-gated compaction (promotion, dreaming, flush); the consented half of the two-compaction contract. → [channels-and-strands.md](channels-and-strands.md#deliberate)
- **Descriptor** — see *PatternDescriptor*.
- **Dreaming sweep** — the consented, curated memory consolidation that flows through the gates as a proposal; contrast auto-compact. → [concepts.md](concepts.md#the-two-compaction-contract)
- **Fail-closed** — every unknown (surface, thread, channel, panic) resolves to `Reject`; a conformance requirement per RFC-0001 §5.4, not a feature. → [authority-model.md](authority-model.md#fail-closed-is-a-conformance-requirement-not-a-feature)
- **Forced (channel)** — runtime-initiated context compaction with no familiar cooperation available; governed by WARD-C1–C6. → [channels-and-strands.md](channels-and-strands.md#forced)
- **Frayed (tension state)** — one strand failed, thread not snapped; repairable in place; MUST surface to the operator; maps to `DegradeToProposal`. → [authority-model.md](authority-model.md#frayed)
- **Gate 4** — RFC-0001 §5.4's final canonical diff check before apply; the real security boundary; MUST NOT be skippable. → [authority-model.md](authority-model.md#fail-closed-is-a-conformance-requirement-not-a-feature)
- **Gate-shaped receiver** — what coven-threads is to the daemon: the typed entry point (`validate`) for identity-surface mutation requests. → [architecture.md](architecture.md#where-coven-threads-sits)
- **Holds (tension state)** — all strands intact; the thread carries its full authority contract; maps to `Permit`. → [authority-model.md](authority-model.md#holds)
- **Loom** — what Ward's four gates are in weave vocabulary: the fixed structure threads run through — gates are not threads. → [concepts.md](concepts.md#thread--an-authority-relationship)
- **ManifestEntry (strand)** — membership in an external, Merkle-rooted hash manifest; the reference point that survives when the context window itself is under threat. → [channels-and-strands.md](channels-and-strands.md#manifestentry)
- **Mutation (channel)** — the default write path: direct client mutation via the daemon; every thread holds under it or the daemon rejects. → [channels-and-strands.md](channels-and-strands.md#mutation)
- **MutationRequest** — one attempted write: `(surface, writer, channel)`; momentary, carries no authority of its own. → [faq.md](faq.md#whats-the-difference-between-a-thread-and-a-mutation-request)
- **PatternDescriptor** — the *derived*, non-authoritative structural summary of a pattern predicate, for humans and tools; MUST NOT be gated on. → [concepts.md](concepts.md#the-descriptor-vs-predicate-anti-pattern)
- **PatternPredicate** — the *authoritative* coherence test over a weave's threads; enforcement gates on the predicate, never the descriptor. → [concepts.md](concepts.md#the-descriptor-vs-predicate-anti-pattern)
- **Permit (verdict)** — thread holds, weave coherent at the surface: mutation applied, verdict audited. → [authority-model.md](authority-model.md#permit)
- **Protected surface** — an identity-bearing file or field (SOUL.md, MEMORY.md, identity fields) that threads bind to as a typed surface. → [concepts.md](concepts.md#thread--an-authority-relationship)
- **Reject (verdict)** — the mutation must not happen; always carries a typed, audit-legible reason. → [authority-model.md](authority-model.md#reject)
- **Repair** — restoring a frayed strand in place (thread returns to Holds); a snapped thread instead requires a fresh authority ceremony. → [authority-model.md](authority-model.md#the-thread-tension-state-machine)
- **Serialization (channel)** — export/import round-trip, format-mediated, Ward-external; governed by WARD-C7. → [channels-and-strands.md](channels-and-strands.md#serialization)
- **SerializationMarker (strand)** — the C7 strand: format version + contract hash committing to the serialization contract; required on `Serialization`-channel threads. → [channels-and-strands.md](channels-and-strands.md#serializationmarker)
- **Signature (strand)** — signing-key commitment: provenance that survives even where hashes could be recomputed. → [channels-and-strands.md](channels-and-strands.md#signature)
- **Snapped (tension state)** — terminal severance; surface read-only until a fresh authority ceremony; maps to `Reject`. → [authority-model.md](authority-model.md#snapped)
- **Strand** — (*fiber inside a thread: hash | signature | manifest entry | audit trail | serialization marker*); a thread survives a channel iff its strands survive that channel. → [concepts.md](concepts.md#strand--fibers-inside-a-thread)
- **Tension** — a thread's standing under load: Holds, Frayed, or Snapped. → [authority-model.md](authority-model.md#the-thread-tension-state-machine)
- **Thread** — (*authority relationship: surface → writer*); one per `(surface, writer)` pair; the object every gate check interrogates. → [concepts.md](concepts.md#thread--an-authority-relationship)
- **Two-compaction contract** — `Deliberate` and `Forced` are distinct channels with distinct survival requirements; invariant #3 of the design. → [concepts.md](concepts.md#the-two-compaction-contract)
- **Verdict** — the gate's answer to a mutation request: `Permit`, `DegradeToProposal`, or `Reject`. → [authority-model.md](authority-model.md#the-three-verdicts)
- **Ward** — RFC-0001 §5: the normative spec (separation, tiers, gates, probes, audit) that coven-threads implements. → [architecture.md](architecture.md#relationship-to-rfc-0001-and-to-ward)
- **`ward.audit`** — the single daemon-owned audit store: an append-only table inside `coven.sqlite3` (spelled `ward_audit` in SQL). → [architecture.md](architecture.md#the-wardaudit-store)
- **WARD-C1–C6** — the six compaction-interlock invariants governing the `Forced` channel; canonical home: `coven-grimoire` Ward Layer Spec Brief §9. → [channels-and-strands.md](channels-and-strands.md#forced)
- **WARD-C7** — survives-serialization: export→import yields equivalent tension state or fails visibly; numbered sibling of C1–C6, canonical in the same §9. → [channels-and-strands.md](channels-and-strands.md#serialization)
- **Weave** — (*enforced pattern of threads across a familiar or Coven*); coherent iff its pattern predicate holds; the primary object Coven Cave will render. → [concepts.md](concepts.md#weave--the-enforced-pattern-of-threads)
- **`weave_hash`** — Merkle commitment over a weave's threads in canonical `(surface_path, writer_id)` order. → [channels-and-strands.md](channels-and-strands.md#manifestentry)
- **Writer** — the authority-side endpoint of a thread: the identity proposing writes to a surface (`WriterId`). → [concepts.md](concepts.md#thread--an-authority-relationship)
